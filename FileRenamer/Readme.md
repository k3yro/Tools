# K3 FileRenamer
## Requirements
- .Net Framework 4.7.2
- A folder full of files
## Regex Syntax
| Syntax   |  Description  |
|----------|:-------------:|
| ^        |  Begin of filename |
| $        |  End of filename   |
| \s       | Whitespace |
|[A-z]     | Find chars A-z |
